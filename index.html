Welcome to a basic static HTML page powered by Azure Web Apps! I created this webpage, so I can learn how to deploy a web app using Microsoft's Azure

I used the free trial subscribtion, so 30 days after its deployment, it will not be live.

The steps that I will follow, to clone this file from my git-repo will be the following:

  1. I opened the Cloud Shell in Azure and I cloned the file using this command: git clone https://github.com/testWebPageAzure.git
  2. I defined my git settings using:
    • git config --global user.email "myEmail"
    • git config --global user.name "myUsername"
  3. To get ready to upload this, I initialized Git and then added and committed my file using the command: git init && git add . && git commit -m "file name"
  4. Then I used the follwing command: git remote add azure my-git-clone-URL
  5. And finally I used the command git push azure master
